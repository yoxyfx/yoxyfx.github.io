PHP 설치 과정(~ING)

1. PHP 설치 파일 다운로드

http://windows.php.net/download 에 접속하여 PHP설치 파일을 다운로드

2. 설치 파일 실행

Install Now를 눌러 설치

3. PHP.ini파일 수정

폴더 안에 php.ini-development 파일을 열어서 내용 수정

라인을 찾아 앞에 ; 표시를 제거
"./" 부분에는 PHP설치경로/ext 형태로 수정

4. Nginx와 PHP 연동 설정

Nginx가 설치된 경로에서 nginx.conf 파일 열어줌

C:\nginx\conf\nginx.conf
기본적으로 설치된 Nginx 파일 안에는 html이라는 기본 루트가 지정

(해당 파일 찾는데 어려움이 있어 아직 해당 단계 진행중인 상태)
