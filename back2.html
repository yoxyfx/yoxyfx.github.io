MySQL 설치 과정(~ING)

1. MySQL Installer 다운로드하기

웹 페이지로 https://downloads.mysql.com/archives/installer/ 이동하여 MySQL Product Archives에서 MySQL Installer를 다운로드

2. MySQL 설치하기

다운로드한 파일을 열어 설치 허용을 해줌
MySQL Installer 창이 나타나면, 제일 먼저 setup type을 설정하는데 "Custom"을 선택하고 Next를 누룸
설치가 완료되면 Next를 눌러 Product Configuration 단계로 이동
Windows Service에서는 체크가 제대로 잘 되어있는지 확인하면 된다. 이외에 Service Name은 MySQL80을 그대로 사용해도 되고, MySQL로 바꾸어 사용해도 좋다. 완료되었다면 Next를 누룸
Start MySQL Workbench after setup를 체크했다면 바로 아래와 같이 워크벤치가 열림

3. 워크벤치 실행

폴더 안에 php.ini-development 파일을 열어서 내용 수정

라인을 찾아 앞에 ; 표시를 제거
"./" 부분에는 PHP설치경로/ext 형태로 수정

4. Nginx와 PHP 연동 설정

Nginx가 설치된 경로에서 nginx.conf 파일 열어줌

C:\nginx\conf\nginx.conf
기본적으로 설치된 Nginx 파일 안에는 html이라는 기본 루트가 지정
