MySQL 설치 과정(워크밴치 설치 완료)

1. MySQL Installer 다운로드하기

웹 페이지로 https://downloads.mysql.com/archives/installer/ 이동하여 MySQL Product Archives에서 MySQL Installer를 다운로드

2. MySQL 설치하기

-다운로드한 파일을 열어 설치 허용으로 전환해줌
- MySQL Installer 창이 나타나면, 제일 먼저 setup type을 설정하는데 "Custom"을 선택하고 Next를 누룸
-설치가 완료되면 Next를 눌러 Product Configuration 단계로 이동
-Windows Service에서는 체크가 제대로 잘 되어있는지 확인하면 된다. 이외에 Service Name은 MySQL80을 그대로 사용해도 되고, MySQL로 바꾸어 사용해도 좋음 완료되었다면 Next를 누룸
-Start MySQL Workbench after setup를 체크했다면 바로 아래 이미지와 같이 워크벤치가 열림
