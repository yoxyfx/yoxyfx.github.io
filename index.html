백엔드 개발자
임승현 입니다

HTML, CSS, JavaScript와 같은 프론트엔드 기술과 함께 최근에는 PHP, MySQL, Python을 학습하여 백엔드 역량을 확장하고 있습니다.
폭 넓은 경험을 바탕으로 다양한 기술을 통합해 복잡한 문제를 해결하는데 주력하고 있습니다.

주요: html, css, javascript, 반응형웹, 웹표준, 웹접근성, MySQL, Python, PHP
보조: photoshop, illustrator, figma, premiere, after effect

MySQL / Python / PHP: 프로그램 설치 및 현재 공부 중.

HTML/CSS: 웹 표준을 지키며 section마다 효율적인 태그를 활용하여 HTML을 구성. keyframes을 사용하여 애니메이션 작업을 할 수 있dma

반응형 웹: 다양한 화면 크기에 맞춰 자동으로 조절되는 레이아웃 설계. 미디어 쿼리를 사용해 데스크탑, 모바일, 태블릿 등의 기기에서 최적화된 웹 디자인을 제공합니다. Flexbox와 Grid 시스템을 활용하여 유연한 레이아웃을 구축한 경험.

JavaScript: 애니메이션 구현(CSS와 JavaScript를 활용한 부드러운 애니메이션), Canvas API 활용(2D 그래픽과 애니메이션 제작), DOM 조작(HTML 요소를 선택하고 추가, 삭제, 수정), 이벤트 처리(클릭, 키보드 입력, 스크롤 등).

Photoshop, Illustrator: 사이트 및 모바일 애플리케이션 디자인에서 그래픽 요소를 제작한 경험. 사용자 인터페이스(UI)와 사용자 경험(UX)을 고려하여 디자인 작업을 진행. 로고, 아이콘, 배너 등의 시각적 요소를 디자인함.

Figma: 협업 도구로써 팀원들과 Figma를 사용하여 협업. 웹 애플리케이션 및 사이트의 UI/UX 디자인을 작성하고, 프로토타입을 통해 동작하는 웹사이트, 앱 프로토타입을 만든 경험.

Github: Git을 사용하여 프로젝트 관리

Premiere, After Effects: 영상 편집 및 모션 그래픽 영상을 구현. 영상 색 및 소리 보정, 자막 넣기.
