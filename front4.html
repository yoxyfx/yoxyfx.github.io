주요 코드

DOMContentLoaded 이벤트:
JavaScript가 DOM에 접근할 준비가 되었음을 알려주기 때문에, 드래그 기능을 활성화하는 코드를 실행 

                
document.addEventListener('DOMContentLoaded', (event) => { ... });

mousedown: 드래그가 시작될 때 마우스 클릭 위치를 캡처하여, 그 위치를 기준으로 요소를 이동시키는 데 사용 

                
draggable.addEventListener('mousedown', (e) => {
    isDragging = true;
    offsetX = e.clientX - draggable.getBoundingClientRect().left;
    offsetY = e.clientY - draggable.getBoundingClientRect().top;
    draggable.style.cursor = 'grabbing'; 
});

mousemove 이벤트: 우스를 움직일 때마다 발생하는 이벤트입니다. 사용자가 마우스를 움직일 때마다 드래그 중인 요소의 위치를 갱신하여 실시간으로 이동
isDragging이 true일 때만 드래그가 진행되도록 조건을 추가하고, e.clientX와 e.clientY를 사용해 새로운 위치를 계산하여 요소를 이동 

                
document.addEventListener('mousemove', (e) => {
    if (isDragging) {
        draggable.style.left = `${e.clientX - offsetX}px`;
        draggable.style.top = `${e.clientY - offsetY}px`;
    }
});

실시간 위치 변경: 요소를 드래그할 때 마우스 커서를 grabbing으로 변경하여 사용자가 드래그 중임을 시각적으로 표시하고, 드래그가 끝나면 다시 기본 커서(move)로 돌아가도록 설정 

                
draggable.style.left = `${e.clientX - offsetX}px`;
draggable.style.top = `${e.clientY - offsetY}px`;
